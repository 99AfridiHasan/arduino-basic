Arduino Bootcamp

A Different Approach.
A hands-on workshop for beginners and intermediate Arduino users that gives emphasis to ditching delays, basic object-oriented programming, and clean readable code.

About Arduino Bootcamp

This course is designed with one thing in mind: to give powerful programming concepts that significantly help with Arduino projects. Some concepts discussed are usually reserved for intermediate users, but this shouldn't be the case.

A Different Approach

This workshop introduces object-oriented programming as the main philosophy to think about designing your Arduino code. It also emphasizes the importance of clean readable code. A more detailed discussion of what sets this workshop apart is linked here<./a>
